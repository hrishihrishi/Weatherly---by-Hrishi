# A Weather Forecast Application built using HTML,Tailwind CSS and JavaScript. 

# Setup & Usage
1. Clone the repository
2. Open the index.html file in a web browser
3. Enter a city name in the search bar and click the search button
4. The weather data will be displayed in the weather information section
5. The 5-day forecast will be displayed in the extended forecast section

Github RepositoryLink : https://github.com/hrishihrishi/Weatherly---by-Hrishi

# Features
* LocalStorage
* Responsive Design
* Error Handling
* Loading States
* Dynamic data fetching using JavaScript
* Search weather by location name
* See current location weather 
* 5 days weather forecast

# Tech Stack
* HTML
* Tailwind CSS
* JavaScript
* OpenWeatherMap API
